Introduction

The absorption spectrum occurs when light is absorbed by matter at specific wavelengths. The emission spectrum is observed when matter emits light at characteristic wavelengths. These phenomena are fundamental in physics and chemistry.

Formula Calculator

Use the formula \( E = h \nu = \frac{hc}{\lambda} \) to calculate the energy of a photon.

Code Execution

Below is an example Python code for the formula:

import math

# Constants
h = 6.626e-34  # Planck's constant in Js
c = 3e8        # Speed of light in m/s

# Input wavelength in nm
wavelength_nm = float(input("Enter wavelength (in nm): "))
wavelength_m = wavelength_nm * 1e-9

# Calculate energy
energy = (h * c) / wavelength_m
print(f"Energy: {energy} J")

To execute this code:

  • Copy the code into a Python editor or IDE.
  • Run the code and input a wavelength when prompted.
